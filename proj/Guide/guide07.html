주석(Java Script 규칙)

  • js 주석은 최대한 자세하게 누가봐도 이해 가능하게 작성하는 것을 지향한다.
  • 해당 기능이 필요한 html 내부 하단에 script 초기 작성을 지향하고 그 이후에 필요 시 front.js으로 코드를 이동한다.
  • 공통적인 script(modal, tab, toggle, gnb…) 는 common.js에 작성한다.
  • Java Script 주석의 기본규칙은 하단 작성내용을 기준으로 하며 기능위주의 주석을 작성한다.

Java Script 주석의 기본규칙

  • 기본 설명은 한 줄 주석 처리 ex)이미지 변수 갯수처리 관련된 주석 
                                //보여지는 이미지의 갯수
                            const imgQtt = 10;
  • 디테일한 설명 주석은 여러 줄 주석 처리 
                                /* 
                                modal open btn click 시 modal이 열리는 기능의 function
                                ...
                                ...
                                ...	(디테일한 설명 추가)
                            */
                            function openModal(){...}
  • 개발 시 상황에따라 변해야하는 부분들 
                                //로그인 정보에따라 썸네일 이미지 변경
                            function loginThumb(getGender, ...){...)

  • 제목 셀마다 각각 다른 id 값을 부여하고, 내용 셀과 관련이 있는 제목 셀의 id 값을 headers에 나열한다.
  • 스크린리더에서는 headers에 나열된 id 값 순서대로 제목 셀을 읽어준다.
  • 즉, 이 "902"를 그냥 "902"라고 읽어주는 것이 아니라, "기준일 2018년 10월 20일 경기도 매매가 면적단가 902"라고 제목들을 연결해서 읽어주는 것이다.
